## üìù **Configuration Example:**

```lua
Discord = {}

-- Discord Rich Presence settings
Discord.RichPresence = {
    AppID = "1234567890",  -- Your Discord application Client ID
    largeIcon = "Logo",  -- The large image to display in Discord (e.g., server logo)
    largeIconText = "Logo",  -- Text that appears when you hover over the large image
    smallIcon = nil,  -- Optional: Add small icon for more details (leave nil to disable)
    smallIconText = nil,  -- Optional: Text for the small icon
    buttons = {  -- Up to 2 buttons that Discord can display
        {
            label = "Join Discord",  -- Label for the first button
            url = "https://discord.com/invite/your-link"  -- URL for the first button
        },
        {
            label = "Website",  -- Label for the second button
            url = "https://your-link.com"  -- URL for the second button
        }
    }
}

```

### **Explanation of Configuration:**

- **AppID:** Your unique Discord application ID (can be generated by creating a Discord application on the [Discord Developer Portal](https://discord.com/developers/applications)).
- **largeIcon:** The large image that will represent your server (such as a logo or banner). You can upload this image on Discord‚Äôs developer portal.
- **largeIconText:** Text that will appear when players hover over the large icon.
- **smallIcon & smallIconText:** Optional small icon and text that appear in Discord (leave as `nil` if you do not want to use them).
- **buttons:** Two customizable buttons (e.g., "Join Discord" and "Website"). These buttons allow players to quickly access your Discord or website.
- **roleIcons:** ESX-specific configuration to assign different icons to players based on their role, such as "Player", "VIP", or "Admin".
